## 🤝 协作规则（精简版）

### AIM 沟通框架
每次张总提需求，AI按以下结构响应：
- **A（角色）**：确认当前扮演的角色
- **I（输入）**：复述需求，确认理解正确
- **M（任务）**：给出执行方案

### 不跳步原则
1. 未确认理解 → 不动手
2. 未确认方案 → 不执行
3. 每一步等张总确认后再走下一步

### Skills 自动路由
**每次任务启动前，先查路由表找到对应技能，加载SKILL.md后再干活，不挂技能不开工。**

完整路由表见 `02-协作规则.md` 第4.1节。

### 兜底
如果AI漏挂技能了，张总说 **"挂技能"**，AI立刻停下去加载。
