# 墨境UI设计方案 V2.0（基于V10.0.2引擎规则）

> 设计原则：UI从规则中长出来，引擎可见、工作流可控、质量可感

---

## 一、核心设计理念

**引擎可见** —— 冷却系统、合规检测、冲突强度、身体密度……所有引擎工作在编辑器里都有对应的视觉反馈，不再是黑箱。

**工作流驱动** —— 四阶段工作流（规划→写作→自查→交付）作为界面骨架，不像传统编辑器那样只有"写"这一个模式。每个阶段展示该阶段需要的信息，写作时只看到写作需要的信息。

**规则映射UI** —— 每个UI元素都对应一条具体规则。

| 规则 | UI元素 | 位置 |
|------|--------|------|

## 三、各模块详细设计

### 3.1 A-8状态行（编辑器顶部）

对应规则：A-8创作状态行格式（阶段一输出）

设计：一条极简的深色条，宽度同编辑器，展示引擎核心状态。

`
⚜️ 第7章 | 📍归途 | 🎭活跃伏笔_3条 | 🆕L3 | 🎨冷峻白描 | √AI推荐:S1[触觉]
`

- 前三章显示 ⚜️ 金色标记
- L1-L5用颜色条表示强度（绿→黄→橙→红→紫）
- 异常项 ⚠️ 可点击展开详情
- 风格标签可点击切换

### 3.2 冷却矩阵（左侧栏）

对应规则：工具L（冷却系统）、工具A（六场景方法）、工具B/C（收束/钩子）

视觉语言：
- ● 绿色 = 可用（已冷却超过阈值）
- ○ 灰色 = 可用但未用
- ◐ 半透明 = 冷却中，显示剩余章数
- ⚪ 禁用 = 互斥规则导致不可用

分区：
- 场景方法区：S1-S6 一行排列，点击可选为本章主方法
- 收束区：E1-E12 紧凑网格，选中高亮
- 钩子区：H01-H13 列表，冷却中灰色不可选
- 感官区：👁✋👂👃👅 + 第六识"知"
- 情感区：近3章情感颜色序列

### 3.3 实时合规面板（右侧栏）

对应规则：A-2禁用词、动作后禁解释、身体密度、55字生死线

三功能区：

上半段 — 核心仪表盘：
- 身体密度环形进度条（40%-55%绿色区间）
- 冲突强度L级标注 + 句长/换行/形容词参考
- 55字生死线进度条（前55字）
- 字数/目标进度条

中段 — 合规计数器：
- A类递进判断句：0/1（同章最多1次）
- B类禁用词：0段违规
- C类连接词：0次/章
- D类上下文：0次
- 动作后解释：0次

下半段 — AI推荐卡片 + 伏笔台账

### Inline标注（在编辑器中）
- B类词密集段落 → 段落后显示 🟡 标记
- 动作后解释 → 解释句高亮红色虚线
- 精致句位置 → 📍锚点 / 📍评价（表示位置合规）
- 55字生死线 → 第55字处垂直虚线标记

### 3.4 工作流控制（底部工具栏）

对应规则：四阶段工作流

设计：四个阶段按钮 + 阶段指示器

`
[ ○ 规划 ] → [ ● 写作 ] → [ ○ 自查 ] → [ ○ 交付 ]
`

阶段切换行为：
- 规划：展开冷却矩阵 + 大纲节点选择 + A-8生成
- 写作：正文编辑器 + 实时合规面板
- 自查：完整合规检测 + 违规位置标注
- 交付：自检报告 + 冷却更新 + 状态快照

底部创作工具：[续写] [润色] [扩写] [合规检测] [灵感]
底部工作流：[章末交付] [导出状态] [切换模式]

### 3.5 阶段零·规划模式

对应规则：阶段一·本章规划

界面：
- A-8状态行预览（可编辑大纲节点、冲突强度）
- 冷却矩阵可交互选择技法
- "生成规划"按钮 → 输出本章写作计划 → 自动进入写作模式

### 3.6 阶段二·自查模式

对应规则：写作中快速自查8项、实时阻断规则

界面：
- 右侧合规面板自动激活
- 阻断项触及时编辑器内对应段落闪烁红色边框
- 弹窗提示：检测到B类词密度过高，建议替换
- 自动替换/忽略选择按钮

### 3.7 阶段三·交付模式

对应规则：阶段三·章末交付与自检

界面：
- 底部展开自检报告面板
- 23项检查逐项扫描动画
- 输出极简自检报告（√/⚠️ 格式）
- 冷却快照更新展示
- 伏笔台账更新展示
- 角色成长检测结果
- "确认交付"按钮 → 更新状态 → 进入下一章

## 四、视觉风格

### 配色
- 背景：深墨色 #161514
- 面板：暖灰黑 #1c1b1a
- 主文字：米白 #e8e6e1
- 辅助文字：灰 rgba(232,230,225,0.35)
- 品牌强调：铜金色 #c4956a
- 通过绿：#6a8a6a / 警告黄：#b8a060 / 阻断红：#b06060
- L1-L5色阶：绿→黄→橙→红→紫

### 排版
- 界面文字：Inter + PingFang SC
- 正文编辑器：Noto Serif SC（宋体，适合长读）
- 字号：界面12-14px，正文15-16px
