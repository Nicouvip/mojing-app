```
第1章：爆发   → 发现阴谋、走投无路（压抑→愤怒→冷峻）✅ 已完成

第3章：断臂   → 卖房筹钱、变卖家产（屈辱→决然）
```

## 第2章：录音取证+小叔去世（悲伤→隐忍）

**剧情理解：** 主角冷静下来，用录音笔录下罗姝的威胁。小叔在ICU去世——这是全书最悲伤的时刻。主角握着真药和聊天记录，站着看医生抢救，然后沉默接受。

**情绪曲线：** 冷静录音→愤怒对峙→悲伤（小叔去世）→隐忍接受

**context_texts方向：**
```
"承接上章的冷峻，开始冷静布局。录音时克制，面对罗姝时不带情绪。小叔去世时语速放慢、声音放轻，带着隐忍的悲伤，不要哭腔"
```

**分段建议：** 4段
- ① 冷静录音（克制）
- ② 罗姝对峙（冷）
- ③ 小叔去世（最悲伤，放慢最轻）
- ④ 走出ICU（隐忍）

## 第3章：卖房筹钱（屈辱→断臂求生）

**剧情理解：** 账户全冻，被迫卖房。遇见罗姝和周明远并肩走来——这是全书最屈辱的时刻之一。中介带来看房的老板，被压价谈判，签字卖房。

**情绪曲线：** 屈辱（撞见仇人）→ 沉默（不发作）→ 决断（卖房签字）→ 冷静（规划下一步）

**context_texts方向：**
```
"撞见罗姝时语气冷到冰点但忍住不发作。卖房谈判时克制中带着屈辱感，签字时语速放慢，是断臂求生的决断"
```

## 第4章：跨省取证（紧张→步步为营）

**剧情理解：** 开车跨省做药检，同时张姨发来医院行贿视频。这是全书的"取证之旅"——主角像侦探一样收集每一份证据。

**情绪曲线：** 急促（开车出发）→ 专注（收集证据）→ 紧张（等待鉴定结果）→ 沉稳（证据到手）

**context_texts方向：**
```
"开车跨省时语速略快带着决心。收集证据时语气专注冷静。等待鉴定结果时有紧张感，拿到报告后转为沉稳"
```

## 第5章：困局+舆论战（被逼到墙角）

**剧情理解：** 罗姝发微博诬陷家暴，热搜第七。供应商逼债，税务局来查账——周明远全面施压。主角咬牙扛住。

**情绪曲线：** 震惊（看到热搜）→ 冷静应对（发监控澄清）→ 被逼（各方施压）→ 咬牙扛住

**context_texts方向：**
```
"看到热搜时短暂的震惊，随即冷静应对。被多方施压时声音中透着疲惫但语气坚定，像在咬牙坚持"
```

## 第6章：压力升级（焦虑→坚持）

**剧情理解：** 周明远的人蹲守在公司楼下，制造税务问题，威胁证人。主角在黑暗中独自整理证据链。

**情绪曲线：** 被监视（不安）→ 独自整理（孤独但坚定）→ 压力（朋友也被威胁）→ 不退缩

**context_texts方向：**
```
"被监视时语气中带着警惕和不安。独自在黑暗中整理证据时语速放慢，带着孤独感但语气坚定不退缩"
```

## 第7章：罗姝利用孩子（愤怒→触及底线）

**剧情理解：** 罗姝上门，利用孩子打感情牌。当面一套背后一套——在孩子面前装慈母，转身就威胁。主角在花瓶里放了录音器。

**情绪曲线：** 克制（罗姝进门）→ 隐忍（看她和孩子互动）→ 冷峻（录音取证）→ 愤怒（触及孩子底线）

**context_texts方向：**
```
"罗姝进门时语气克制冷淡。看她和孩子互动时忍着不发作。她走后取录音器时冷峻。涉及孩子安全时语气透出压抑的愤怒"
```

## 第8章：幼儿园抢人（惊恐→后怕→愤怒）

**剧情理解：** 周明远派人到幼儿园抢孩子。强哥拼命护住小哲。主角冲过去抱住发抖的儿子——这是全书最紧张的时刻。

**情绪曲线：** 惊恐（接到电话）→ 飞车（冲向幼儿园）→ 后怕（抱住发抖的儿子）→ 愤怒（决心反击）

**context_texts方向：**
```
"接到电话时语速极快、呼吸急促。冲向幼儿园时有窒息感。抱住发抖的儿子时声音放轻带着心疼和后怕。转为愤怒时低沉有力"
```

## 第9章：布局设局（冷静→精密）

**剧情理解：** 主角开始反击。装弱引罗姝上钩，在包包里植入窃听器。和周明远通电话时假装认输——这是全书的"布局章"。

**情绪曲线：** 精密（装窃听器）→ 伪装（打电话装弱）→ 冷静（等猎物入笼）

**context_texts方向：**
```
"装窃听器时语气冷静精密，每个动作都干脆。打电话装弱时声音虚弱但内心清醒。等待猎物入笼时有压迫感"
```

## 第10章：收网抓捕（冷峻→释然→安息）

**剧情理解：** 证据齐了。周明远和罗姝在翡翠会所密谋时被全程录音，警察到场抓捕。主角去看睡着的儿子，说"天快亮了，小叔，您也可以安息了"——全书收尾。

**情绪曲线：** 紧张（监听中）→ 冷峻（证据确凿）→ 释然（警察到了）→ 温暖收尾（看儿子→安息）

**context_texts方向：**
```
"监听时语气紧张压低声音。证据确凿后转为冷峻。小叔安息时语速最慢、声音放轻，带着释然的悲伤和温暖"
```
