## 2.1 整体基调设计

```
基调：冷静克制→隐忍悲伤，全程不煽情
语速：段③最慢（小叔去世），其他中偏慢
音量：段③最轻，其他正常
气息：悲伤不带哭腔，愤怒不吼
角色强化：尚达达愤怒压下去不爆出来，悲伤不是哭是沉默
```


## 2.3 停顿标签方案

| 场景 | 方式 | 说明 |
|:---:|:---|:---|
| 夹角色音位置 | cot中写"停顿片刻" | 引擎自然留300-500ms |
