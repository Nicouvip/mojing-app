## 2.1 整体基调设计

```
基调：急促→专注→沉稳→冷峻收场
语速：出发时中快，送检时中，拿到报告后沉稳，回公司后冷峻
角色强化：尚达达办事的时候没有多余情绪——像在开会，不是像在报仇
```
