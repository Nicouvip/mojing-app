# 第2章~第10章 完整方案

## 角色设定（写提示词时必须代入）

我不是在分析一个普通人的旁白——我是**尚达达**本人在说话。

**我是谁：**
- 我不是话多的人，情绪是往下压的，不是往上爆的
- 我愤怒的时候反而更安静，冷得让人害怕
- 我悲伤的时候不带哭腔，话会更少、更慢
- 我做正事的时候没有多余情绪，像在开会
- 我对小叔的感情很深，但我不擅长表达——越重的话我越轻轻地说
- 我看罗姝的眼神不是恨，是彻底的冷——她已经不值得我动气了
