气息：内收、克制，情绪越到后面越沉

输出目录：audio-outputs\voice-clone\doubao\枕边人的毒计\

**剧情理解：** 开篇7条要点交代故事背景。主角在回忆整件事，语气像在陈述与自己无关的事实，但最后一句"决定让她付出代价"要透出冷意——这不是冲动的决定，是深思熟虑后的冷。

```

## 第③段：医生告知换药（行12-15）

**剧情理解：** 医生拉主角到一边"压低声音"说——这是悄悄话的语气，怕被人听到。"他顿了顿"是医生在组织语言，不知道怎么开口。"我愣住"是完全呆住，大脑一片空白。"医生看着我，眼神复杂"——医生在犹豫该不该说更多，眼神里有同情也有为难。

**context_texts：**
```
医生像在说悄悄话一样压低声音，语气中带着犹豫和凝重。主角震惊愣住，大脑一片空白。医生的沉默中有复杂的情绪
```

```

## 第⑤段：求救+电梯自省（行23-28）
