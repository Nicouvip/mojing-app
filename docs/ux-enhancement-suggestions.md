## 心流体验增强建议

### 1. 编辑器沉浸模式（Focus Mode）

```
触发方式：工具栏「沉浸」按钮 或 F11
效果：
  - 侧栏折叠为 icon-only（64px）
  - 工具面板完全隐藏
  - 编辑器背景扩大至全宽
  - 顶部导航栏渐隐（鼠标移到顶部恢复）
  - 窗口标题切换为当前章节名
退出：Esc 或再次点击按钮
```

### 2. 侧栏交互优化

```
当前问题：
  - 侧栏项目列表缺少 hover/active 视觉反馈
  - 拖拽排序缺少视觉指示器

建议方案：
  - 选中项加左侧 3px primary 色条 (border-l-3 border-l-primary)
  - 未选中项 hover 时显示微弱的 bg-sidebar-hover
  - 拖拽时显示虚线占位框 (border-2 border-dashed border-border rounded-md)
  - 侧栏项目支持缩进层级表示章节结构
```

### 3. 工具栏精简

```
当前工具栏在编辑器顶部，建议：
  - 默认只显示最常用的 5 个操作（加粗、斜体、标题、引用、撤销/重做）
  - 二级操作放在「...」更多菜单中
  - 可配置「固定工具栏」和「自动隐藏工具栏」两种模式
```

### 4. 输入反馈

```
- 字数统计：实时更新，每 500 字显示一个轻提示点
- 自动保存：输入停顿 2s 后触发保存，状态显示「已保存 ✓」/「保存中 ⟳」
- 打字音效：可选功能，每个按键触发极短的 click 音（类似打字机）
  （这是可选趣味功能，默认关闭）
```

### 5. 色彩无障碍

```
所有文字色与背景色的对比度 ≥ 4.5:1（WCAG AA）
特别检查项：
  - muted-foreground (#8E929B) 在 background (#f7f8fa) 上的对比度
  - sidebar 文字与背景对比度
  - 暗色主题下的所有文字对比度
```
