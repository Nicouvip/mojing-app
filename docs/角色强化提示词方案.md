# 角色强化提示词补充方案

## 核心原则

旁白是"尚达达"在说话。他的性格决定了语气走向：

**冷静理性** — 愤怒往下压，不是往上爆。越愤怒语气越平静得可怕  
**简练干脆** — 不拖泥带水，不煽情不夸张  
**内敛克制** — 悲伤不哭腔，愤怒不嘶吼，冷到极致反而轻笑  

---

## 第2章：录音取证+小叔去世

### 关键句cot强化

| 原文 | 普通写法 | 尚达达写法 |
|:---|:---|:---|
| "我气得深吸了一口气" | 愤怒地深呼吸 | 深吸一口气，把怒气压下去，再缓缓呼出 |
| "我接过协议，翻得很慢" | 慢慢地翻协议 | 像在审合同一样冷静地翻，实际在拖延时间 |
| "我冷漠地抬眼盯着她" | 冷冷地盯着 | 懒得动气，像看陌生人一样抬眼看她 |
| "他眼睛还睁着，不动了" | 悲伤地说 | 声音没有任何波动，但说不动了三个字时微微停顿 |
| "我站着没动" | 呆站着 | 不是呆住，是太清醒了以至于不知道下一步该做什么 |
| "我伸手，轻轻合上他眼皮" | 温柔地说 | 动作很轻但语气克制冷静，像在处理一件必须做的事 |

### context_texts角色强化
```
"用冷静得可怕的语气录音取证，愤怒是压在底下的。
小叔去世时不带哭腔，每个字都像从喉咙里挤出来。
走出ICU后语气恢复冷静，但透出隐隐的冷意"
```

## 第3章：卖房筹钱

### 关键句cot强化

| 原文 | 尚达达写法 |
|:---|:---|
| "我看着他们，没说话" | 不怒不躁，就这么看着，把他们的笑容记在心里 |
| "我转身走进电梯" | 动作干脆，不多说一个字 |
| "我的脸冷得像冰" | 不是愤怒，是彻底冷下来的脸，像戴了面具 |
| "我站在机器前" | 没有崩溃没有砸机器，就这么站着，脑子在飞快转 |
| "我拿出房本" | 像递一张名片一样平静，内心在滴血但面上不露 |

### context_texts角色强化
```
"撞见仇人时克制住不发作——尚达达不是当场翻脸的人。
卖房签字的每一个动作都干脆利落，内心在滴血但面上不露。
这不是冲动是冷静到可怕的断臂求生"
```

## 第4章：跨省取证

### 关键句cot强化

| 原文 | 尚达达写法 |
|:---|:---|
| "我拿出手机，翻拍那半板真药，正反两面，特写" | 像警察拍证物一样冷静、专业、一丝不苟 |
| "我保存，备份" | 短短四个字，字字干脆，没有多余情绪 |
| "我坐在走廊长椅上等待" | 不是焦虑地等，是安静地、耐心地等 |
| "我把报告折好，放进内袋" | 动作珍重但表情冷静 |

### context_texts角色强化
```
"像侦探一样冷静取证，每一件事都做得很专业。
不慌不忙，不喜不悲——尚达达在办正事时是完全没有情绪的。
只有把报告放进内袋时有一丝珍重"
```

## 第5章：困局+舆论战

### 关键句cot强化

| 原文 | 尚达达写法 |
|:---|:---|
| "我登录公司官网，上传监控视频" | 被污蔑后不吵不闹，直接上证据，干脆利落 |
| "我始终沉默，收拾碎片，转身离开" | 这里最妙——整个家暴现场他始终没说话，沉默是最有力的反击 |
| "我看着黑屏的手机" | 不是发呆，是脑子在规划下一步 |

### context_texts角色强化
```
"被污蔑时不出声不辩解，直接甩证据——这是尚达达的方式。
沉默比任何反驳都有力。
收拾碎片时动作冷静，像在处理一件与己无关的事"
```

## 第7章：罗姝利用孩子

### 关键句cot强化

| 原文 | 尚达达写法 |
|:---|:---|
| "我弯腰抱他" | 抱起儿子时语气自然的温暖，和之前的冷形成对比 |
| "我走到花瓶前，调整角度" | 动作自然，不能让罗姝察觉——像在摆弄花瓶一样随意 |
| "她冷笑，走到客厅沙发坐下" | 旁观者视角，不带情绪地叙述——但越没情绪越可怕 |

### context_texts角色强化
```
"在儿子面前是正常的父亲语气，温暖自然。
面对罗姝时恢复冷静，像在演一出戏。
取录音器时动作随意但内心冰冷——这是尚达达的方式"
```

## 第8章：幼儿园抢人

```
愤怒不是吼出来的，是握紧栏杆时手上暴起的青筋"
```

## 第9章：布局设局

### 关键句cot强化

| 原文 | 尚达达写法 |
|:---|:---|
| "我压低声音。装出虚弱" | 不是真的虚弱，是故意压低声音演戏 |
| "猎物要进笼了" | 冷到极致的自言自语，没有兴奋只有冷静 |
| "我回复一个字：好" | 一个字，干脆利落，不多说 |

### context_texts角色强化
```
"装弱时语气刻意放低但不要演过头——尚达达即使装弱也有分寸。
'猎物要进笼了'这句话要冷到让人起鸡皮疙瘩。
回复短信时一个字都不多说——这是尚达达的风格"
```

## 第10章：收网抓捕

### 关键句cot强化

| 原文 | 尚达达写法 |
|:---|:---|
| "我按下停止键，保存文件" | 动作干脆，没有兴奋——这是预料之中的结果 |
| "备份到三个不同云端" | 严谨、不留漏洞，像在做一次普通的备份 |
| "我放下手机。走到小哲房间" | 语速放到最慢，脚步声放轻，像怕惊醒什么 |
| "天快亮了，小叔，您也可以安息了" | 全书的最后一句话，不是悲伤的哭腔，而是一句平静的陈述——像一个儿子终于完成了父亲的遗愿 |

### context_texts角色强化
```
"收网时没有兴奋没有激动——尚达达不是那种人。
'备份到三个不同云端'语气平淡得像在说今天天气不错。
最后走到儿子房间时语速放到最慢声音放到最轻。
'小叔，您也可以安息了'——平静地、轻轻地、像在睡前说晚安"
```
