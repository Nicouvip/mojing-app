## 第2章：录音取证 → 小叔去世

### 2.1 整体基调设计

```
题材类型：都市现实/商战
基调：冷静→克制→隐忍悲伤，全程不煽情
语速：段落③最慢，其他中偏慢
音量：段落③最轻，其他正常
气息：悲伤时不带哭腔，愤怒时不吼
```

### 2.2 分段语气设计

**段①：冷静录音→对峙**
```
context_texts: "我用录音笔录下她说的每一句，语气冷静克制。我是尚达达，愤怒是压下去的，不是爆出来的"
cot: <cot text=像在签合同一样自然地翻协议,实际在拖延时间>我接过协议，翻得很慢。左手伸进裤袋，摸到手机，指纹解锁，划开录音键</cot>
     <cot text=懒得动气,像看陌生人一样>我冷漠地抬眼盯着她</cot>
     夹角色音×2 → 各留300ms
```

**段②：小叔临终→张姨揭发**
```
context_texts: "小叔在ICU里用最后的力气说话，我握住他的手。张姨拿出证据时语气急促慌乱"
cot: <cot text=握住枯瘦的手,声音放轻>我握住他枯瘦的手</cot>
     <cot text=急促,慌乱>张姨突然从角落冲过来，她一直默默守在旁边</cot>
     夹角色音×3 → 留300-500ms
```

**段③：小叔去世（全章最悲伤处）**
```
context_texts: "小叔走了。我不带哭腔，话更少更慢——尚达达悲伤的时候不是哭，是沉默"
cot: <cot text=声音没有波动,但"不动了"三个字之间微顿>他眼睛还睁着，不动了</cot>
     <cot text=不是呆住,是太清醒了>我站着没动</cot>
     <cot text=动作很轻但语气克制>我伸手，轻轻合上他眼皮</cot>
     夹角色音×2 → 留500ms
```

**段④：处理后事→计划下一步**
```
context_texts: "走出ICU，我恢复冷静。手里攥着证据，心里已经在想下一步怎么走"
cot: <cot text=语气恢复冷静,但透出隐隐的冷意>我走出ICU</cot>
     夹角色音×3 → 留300ms
```

## 第3章：卖房筹钱

### 2.1 整体基调设计

```
基调：屈辱但克制，断臂求生也站直了断
语速：卖房签字时放慢，其他中速
气息：不自怜不叹气
```

### 2.2 分段语气设计

**段①：撞见仇人**
```
context_texts: "我不是当场翻脸的人。撞见他们笑，我不发作——我记住就行了"
cot: <cot text=不怒不躁,就这么看着>我看着他们，没说话</cot>
     <cot text=不是愤怒,是彻底冷下来了>我转身走进电梯。镜面里，我的脸冷得像冰</cot>
```

**段②：ATM全冻**
```
cot: <cot text=没有崩溃,站着脑子在转>我站在机器前，屏幕的光映在脸上</cot>
     <cot text=简短,不拖泥带水>我站在街边，打电话</cot>
```

**段③：卖房签字**
```
cot: <cot text=像递名片一样平静>我拿出房本</cot>
     <cot text=短促有力>签字，按手印</cot>
```

## 第4章：跨省取证

### 2.1 整体基调设计

```
基调：像侦探一样冷静办事，没有多余情绪
语速：开车时中快，等待时慢，拿到报告后沉稳
```

### 2.2 分段语气设计

```
context_texts: "我做事的时候不喜欢说话。拍证据、送鉴定、等报告——每一样都做到位。我不是在发泄情绪，我是在办事"
cot关键句：
  <cot text=动作干脆利落>车钥匙掏出来，解锁，拉车门，坐进去，引擎轰响</cot>
  <cot text=像警察拍证物>翻拍那半板真药，正反两面，特写</cot>
  <cot text=安静地等,不是焦虑>我坐在走廊长椅上等待</cot>
  <cot text=动作珍重但语气平淡>我把报告折好，放进内袋</cot>
```

## 第5章：困局+舆论战

### 2.1 整体基调设计

```
基调：不吵不闹直接上证据，沉默是最有力的反击
语速：中偏慢，不紧不慢
```

### 2.2 分段语气设计

```
context_texts: "她不就是要搞我吗。搞吧——我不吵不闹，我直接上证据。收拾碎玻璃的时候我没说话——我越安静她越慌"
cot关键句：
  <cot text=直接动手做事,没有惊慌>我登录公司官网，上传监控视频</cot>
  <cot text=沉默比任何反驳都有力>我始终沉默，收拾碎片，转身离开</cot>
  <cot text=疲惫但语气坚定>我站在窗边。楼下，两个戴鸭舌帽的男人在对面街口晃悠</cot>
```

## 第7章：罗姝利用孩子

### 2.1 整体基调设计

```
基调：在儿子面前正常父亲语气，面对罗姝时恢复冷静，取录音器时自然到罗姝没察觉
语速：中偏慢
```

### 2.2 分段语气设计

```
context_texts: "她在儿子面前演慈母，我不拆穿——我等她走。她动我可以，动我儿子不行"
cot关键句：
  <cot text=自然的温暖>我弯腰抱他</cot>
```

## 第8章：幼儿园抢人

### 2.1 整体基调设计

```
基调：话少→抱紧→冷静善后→最后愤怒
语速：接电话时快，抱住儿子时慢，善后时中速
```

### 2.2 分段语气设计

```
context_texts: "我冲出会议室的时候什么都没说——我着急的时候话更少。抱住儿子的时候没什么话说，就是抱紧。他们动我儿子，那就别怪我了"
cot关键句：
  <cot text=话少,直接冲>我冲出会议室</cot>
  <cot text=没有安慰的话,只是抱紧,声音放轻>小哲扑进我怀里。身体还在抖</cot>
  <cot text=握紧但脸上没表情>我握紧栏杆</cot>
```


### 2.2 分段语气设计

```
context_texts: "我等这一天很久了。装窃听器的时候手很稳——我练过。打电话装弱的时候声音故意放低。'猎物要进笼了'——这句话我在心里说过很多次了"
cot关键句：
  <cot text=动作精密冷静>我拆开锁扣，植入录音器，磁铁吸附在内壁。重新装好，看不出痕迹</cot>
  <cot text=故意放低,装出虚弱>我压低声音。装出虚弱</cot>
  <cot text=冷到极致不带兴奋>猎物要进笼了</cot>
  <cot text=一个字,干脆>我回复一个字：好</cot>
```

## 第10章：收网抓捕

### 2.1 整体基调设计

```
基调：不激动→不兴奋→轻轻收尾
语速：段落④最慢最轻
气息：全书的最后一口气，是呼出来的，不是吸进去的
```

### 2.2 分段语气设计

```
context_texts: "按下停止键的时候我没激动——我知道迟早会有这一天。打电话报警的时候语气很平静。最后走到儿子房间——天快亮了，小叔，您也可以安息了"
cot关键句：
  <cot text=不兴奋,预料之中>我按下停止键，保存文件</cot>
  <cot text=平静得像说今天天气不错>备份到三个不同云端</cot>
  <cot text=声音放到最轻,像怕惊醒>我放下手机。走到小哲房间。他睡得正熟，小手握成拳头</cot>
  <cot text=轻轻地说,像在睡前说晚安>天快亮了，小叔，您也可以安息了</cot>
```
