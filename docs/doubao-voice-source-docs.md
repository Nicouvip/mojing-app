## 文档一：声音复刻2.0最佳实践

**来源：** https://docs.volcengine.com/docs/6561/2298705?lang=zh

### 核心要点

1. **Prompt（训练音频）要求：**
   - 长度14~30秒，WAV格式
   - 低噪声、单人、单轨音频
   - 情绪尽可能一致，不要大起大落
   - 也不要过于平淡、避免发音模糊、语气生硬
   - 降噪可提升清晰度，但会损失一定相似度
   - 中英混情况，prompt中最好能同时覆盖中英文

2. **context_texts 用法：**
   - 一句话指令提升情感效果
   - 通用合成：`["用最悲伤的语气演绎下面这句话："]`
   - LLM对话场景：把上文query放进去
   - 2.0模型不会完全贴合prompt情感，会更贴合语义信息

3. **情感稳定性：**
   - 情感平稳的prompt → 合成时更稳定
   - 情感丰富的prompt → 多次合成会有变化
   - 想要prompt的音色+情感 → 常驻 context_texts（带上情感描述）

4. **高表现力情感复现：**
   - 不带context_texts → 默认情感
   - 带context_texts → 情感明显提升
   - 示例效果对比：文档中提供了"悲伤"和"生气"两组对比音频

## 文档二：语音指令与标签

**来源：** https://www.volcengine.com/docs/6561/1871062?lang=zh

### 1. 什么是语音指令
控制整体情绪（悲伤/生气）、方言（四川话/北京话）、语气（撒娇/暧昧/吵架/夹子音）、语速快慢、音调高低等。

### 2. 语音指令示例库

| 指令 | 效果 |
|:---|:---|
| "你得跟我互怼！就是跟我用吵架的语气对话" | 吵架、激烈 |
| "用asmr的语气来试试撩撩我" | 暧昧、悄悄话 |
| "用试探性的犹豫、带点害羞又藏着温柔期待的语气说" | 复杂情感 |
| "用低沉沙哑的语气、带着沧桑与绝望地说" | 沧桑、绝望 |
| "用颤抖沙哑、带着崩溃与绝望的哭腔，夹杂着质问与心碎的语气说" | 崩溃哭腔 |
| 无指令 vs 有指令效果对比 | 效果差异显著 |

### 3. 引用上文
输入合成文本的上文（只引用不合成），模型会理解并承接语境的情绪进行合成。

**示例：**
| 引用上文 | 效果 |
|:---|:---|
| "你怎么评价北京这个城市？" | 模型呈现思考和停顿感 |
| "是你吗？怎么看着好像没怎么变啊？" | 重逢的激动语气 |
| "挺好的…就是去年整理旧书，翻到你给我写的毕业留言…" | 温暖的回忆语气 |

## 文档三：V3 SSE API 参数参考

**来源：** https://www.volcengine.com/docs/6561/1598757

### 请求体完整参数

```json
{
  "user": {"uid": "reasonix"},
  "req_params": {
    "text": "合成文本",
    "speaker": "音色ID",
    "model": "seed-tts-2.0-expressive",
    "audio_params": {
      "format": "wav",
      "sample_rate": 24000
    },
    "additions": "{...}"  // JSON字符串！
  }
}
```

### additions 支持的全部参数

| 参数 | 类型 | 范围 | 说明 |
|:---|:---:|:---:|:---|
| speech_rate | int | -50~100 | 语速 |
| loudness_rate | int | -50~100 | 音量 |
| post_process.pitch | int | -12~12 | 音调 |
| context_texts | string[] | 自然语言 | 语音指令（仅expressive） |
| use_tag_parser | bool | true/false | cot标签解析（仅expressive） |
| enable_subtitle | bool | true/false | 字幕/时间戳 |
| silence_duration | int | 0~30000ms | 句尾静音 |
| aigc_watermark | bool | true/false | AIGC水印 |
