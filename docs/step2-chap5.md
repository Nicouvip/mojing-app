## 2.1 整体基调设计

```
基调：被污蔑→冷静应对→被逼到墙角→咬牙扛住
语速：中偏慢，不紧不慢。越被逼越冷静
角色强化：尚达达被污蔑时不吵不闹直接上证据。沉默是最有力的反击
```
